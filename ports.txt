# Notes
# - http_path is a uniform template for host-config generation; real routes may be /health, /api/v1/..., etc.
# Notes (collisions / dual definitions — resolve in host-config.yml / compose)
# - 8120: admin-ui-backend and database-overlord — use distinct hostnames; same internal port.
# - 8099: node-management-staging and gui-hardware-manager — remap host publish if co-hosted.
# - 8090: session-recorder and rdp-server-manager-http — remap host publish if co-hosted.
# - 8080: api-gateway and tron-payment-service — remap host publish if co-hosted.
# - 8092: payout-router and session-merkle-tree-builder — remap host publish if co-hosted.
# - 8600: node-worker and consul-lan-serf — remap host publish if co-hosted.
# - 8101: rdp-session-controller and tron-client-aux — remap host publish if co-hosted.
# - RDP: configs/services/rdp-services.yml uses 8001 for rdp-server-manager; Dockerfile uses 8090.
# - gui-tor-manager HTTP is 8097; 9050/9051 are Tor socks/control (not the GUI service port).
# - 8097: gui-tor-manager and payment-system-gateway — distinct hostnames; remap host publish if co-hosted.
# - 8098: tron-relay and gui-docker-manager — remap host publish if co-hosted.
# - lucid-server-gateway (server-gateway): same HTTP port as api-gateway (8080); image CMD may bind
#   hostname api-gateway — align container hostname / uvicorn --host with compose (see Dockerfile.server-gateway).
# - node-system-gateway: HTTP 8640; /health local; /app/*→api-gateway:8080; /admin/*→admin:8155/stripped
#   (connections: infrastructure/containers/node/config/node-system-gateway.connections.json;
#   TLS client policy: infrastructure/containers/node/config/openssl-api.yml).
# - chain-to-pay: HTTP 8114; /health local; /app → tron-payment-service:8080; /app/data-chain|consensus|session-chunk
#   → rewritten to /app on upstream (see infrastructure/containers/blockchain/config/chain-to-pay.connections.json).
# - admin-system-gateway: primary HTTP health 8155 (/health). Portal TCP listens (defaults in
#   infrastructure/containers/admin/config/admin-system-gateway.connections.json): 28080→api-gateway:8080,
#   28120→admin-ui-backend:8120, 28050→admin-params-registry:8050, 27000→lucid-tunnel-tools:7000.
#   Override via .env.admin-system-gateway* / mounted connections.json.
# =============================================================================

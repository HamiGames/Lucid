QUICK START GUIDE
===============================================================

For GUI Users:
Step 1: bash gui-api-bridge/scripts/gui-user-setup.sh --full
Step 2: bash gui-api-bridge/scripts/os-detector-linux-enforcer.sh
Step 3: bash gui-api-bridge/scripts/gui-user-setup.sh --check
Step 4: gui-exec health

For Operators:
Step 1: bash gui-api-bridge/scripts/build.sh
Step 2: bash gui-api-bridge/scripts/deploy.sh
Step 3: gui-exec status
Step 4: gui-exec logs

Available Commands:
- gui-exec health          (Check API health)
- gui-exec logs            (Stream logs)
- gui-exec status          (Show status)
- gui-exec stats           (Container stats)
- gui-exec api GET <path>  (Make API call)
- gui-exec exec <cmd>      (Execute command)
